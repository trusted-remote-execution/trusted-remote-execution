Trusted Remote Execution (Rex)

Secure the environment, not the agent.

Script (Rhai)
// Read a file let content = cat("/tmp/rex-example/config.ini"); content // Write to a file write("/tmp/rex-example/config.ini", "level = info");
Policy (Cedar)
// Policy managed by service owner permit( principal, action in [ file_system::Action::"open", file_system::Action::"read", // --- Uncomment below to allow write operations --- // file_system::Action::"write" ], resource ) when { resource in file_system::Dir::"/tmp/rex-example" };
Output
[server] host = 0.0.0.0 port = 8443 workers = 8 error: Permission denied: file_system::Action::"write" for file_system::File::/tmp/rex-example/config.ini
✅ cat() read the file. The policy permits read, so Cedar allowed it.
❌ write() was blocked. The policy permits open and read — write is not listed.

Secure the environment, not the agent.
